# debug_unreachable

> unreachable!() in debug, std::intrinsics::unreachable() in release.

## Usage

Use the crates.io repository; add this to your `Cargo.toml` along
with the rest of your dependencies:

```toml
[dependencies]
debug_unreachable = "^0.0.0"
```
